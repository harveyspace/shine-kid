## 一、产品概述

### 1.1 产品定位

**闪光少年** 是一款面向青少年的AI运动能力分析平台，通过拍摄运动视频，AI自动分析运动能力，生成六边形能力报告，记录孩子成长轨迹，发现运动天赋。

### 1.2 核心价值

| 角色 | 价值 |
|-----|------|
| 孩子 | 看到自己的进步，保持运动热爱 |
| 家长 | 了解孩子运动能力，科学陪伴成长 |
| Scout/球探 | 高效筛选人才，查看客观数据 |

### 1.3 目标用户

- **主要用户**：6-18岁青少年（可独立使用）及其家长
- **次要用户**：体育老师、教练
- **未来用户**：星探、球探、俱乐部

### 1.4 用户模型

用户即运动者本人，无需区分家长/孩子。用户可以：
- 直接注册使用
- 记录自己的运动数据
- 查看自己的能力报告
- 分享报告给家长/教练

### 1.4 产品口号

> "让每个孩子都能闪耀"

### 4.3 六边形能力报告

#### 跳绳能力维度：
| 维度 | 评估指标 |
|-----|---------|
| ⚡ 爆发力 | 腾空高度、启动速度 |
| 🏃 速度 | 平均BPM、最高BPM |
| 💪 耐力 | 持续时间、疲劳指数 |
| 🔄 协调性 | 动作节奏稳定性 |
| 🧘 柔韧性 | 关节活动范围 |
| ⚖️ 稳定性 | 重心控制、平衡 |

#### 足球能力维度（符合主流青训体系）：
| 维度 | 评估指标 | 青训关注点 |
|-----|---------|-----------|
| ⚡ 爆发力 | 冲刺速度、启动爆发力、变向加速 | 短距离冲刺能力 |
| 🏃 速度 | 带球速度、无球跑动速度、反应速度 | 攻防转换速度 |
| 🎯 精准 | 传球成功率、射门准确率、传中精度 | 技术执行力 |
| 💪 耐力 | 跑动距离、高强度跑占比、恢复能力 | 全场比赛续航 |
| 🧘 柔韧 | 髋关节活动度、身体协调性 | 动作舒展度 |
| 🧠 球商 | 视野宽度、决策速度、位置感 | 阅读比赛能力 |

**青训专项指标**：
- 控球能力：停球质量、控球时间、护球成功率
- 传球能力：短传/长传准确率、直塞成功率
- 射门能力：射门力量、射正率、门前终结
- 防守能力：抢断成功率、拦截位置、卡位意识

## 七、风险与对策

| 风险 | 影响 | 对策 |
|-----|------|-----|
| AI计数精度不足 | 用户体验差 | 端侧+云端双重校验 |
| 视频上传慢 | 用户流失 | 压缩算法 + 进度提示 |
| 商标风险 | 品牌受限 | 备选名字：闪光少年AI |
