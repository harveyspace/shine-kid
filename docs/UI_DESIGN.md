> 版本：v1.0

## 一、设计理念

### 1.1 品牌定位

**闪光少年** - 让每个孩子都能闪耀

- **品牌色**：阳光橙 + 天空蓝
- **品牌感觉**：活力、温暖、成长、希望
- **目标用户**：青少年家长，关爱孩子成长

### 1.2 设计关键词

| 关键词 | 视觉表现 |
|-------|---------|
| 活力 | 明快的橙色调 |
| 成长 | 上升的曲线动画 |
| 闪耀 | 金色的光点/粒子效果 |
| 运动 | 动态的曲线和形状 |
| 温暖 | 圆润的边角，柔和的阴影 |

## 八、动画规范

### 8.1 动画原则

| 原则 | 说明 |
|-----|------|
| 自然 | 符合物理运动规律 |
| 快速 | 响应时间 < 100ms |
| 流畅 | 60fps |
| 有意义 | 动画传达信息 |

### 8.2 动画时长

```
瞬间:      100ms  - 按钮反馈
快速:      200ms  - 小元素
标准:      300ms  - 页面转场
慢速:      400ms  - 大元素
特殊:      600ms  - 六边形绘制

曲线:
├── 标准:    Curves.easeInOut
├── 进入:    Curves.easeOut
├── 退出:    Curves.easeIn
├── 弹性:    Curves.elasticOut
└── 回弹:    Curves.bounceOut
```

### 8.3 关键动画

#### 1. 六边形绘制动画
```
动画:     从中心点向外展开
时长:     600ms
曲线:     Curves.easeOut
效果:     描边动画，从0%到100%
```

#### 2. 数字滚动动画
```
动画:     数字从0滚动到目标值
时长:     800ms
曲线:     Curves.easeOut
效果:     千分位分隔符滚动
```

#### 3. 卡片出现动画
```
动画:     从下向上滑入 + 淡入
时长:     300ms
曲线:     Curves.easeOut
间隔:     每个卡片50ms
```

#### 4. 闪光粒子效果
```
动画:     星星从中心向外扩散 + 闪烁
时长:     1000ms
曲线:     Curves.easeOut
粒子数:   8-12个
```

#### 5. 页面转场
```
类型:     左右滑动 / 上下滑动
时长:     300ms
曲线:     Curves.easeInOut
效果:     共享元素放大
```

## 十一、动效示意

### 11.1 六边形绘制动画

```
时间轴:
0ms    → 雷达图中心点出现
100ms  → 中心点发出第一条线（爆发力）
200ms  → 第二条线（速度）
300ms  → 第三条线（耐力）
400ms  → 第四条线（协调性）
500ms  → 第五条线（柔韧性）
600ms  → 第六条线（稳定性）
600ms  → 填充区域渐变显示
800ms  → 数值标签淡入
1000ms → 完成
```

### 11.2 数字滚动效果

```
计数显示:
0ms    → 显示 0
100ms  → 显示 45
200ms  → 显示 89
300ms  → 显示 127
400ms  → 显示 150
500ms  → 显示 150 (停止)

BPM显示:
0ms    → 显示 --
200ms  → 显示 98
400ms  → 显示 115
600ms  → 显示 120
800ms  → 显示 120 (停止)
```

### 11.3 页面转场

```
从首页 → 跳绳录制:
进入动画:     从右侧滑入
退出动画:     保持不动
共享元素:     跳绳卡片放大至全屏
时长:         300ms
曲线:         Curves.easeInOut
```

## 十二、响应式设计

### 12.1 断点

```
手机:     < 390px   (iPhone SE / 小屏)
标准手机: 390-429px (iPhone 12/13/14)
大屏手机: 430-500px (iPhone Plus / Pro Max)
平板:     > 500px
```

### 12.2 适配策略

```
小屏手机:
├── 减小卡片内边距: 16px → 12px
├── 减小字号:      Body 17pt → 15pt
└── 隐藏次要信息

大屏/平板:
├── 增加卡片圆角:  16px → 20px
├── 增加间距:      md 16px → 24px
└── 水平排列卡片:  2列布局
```
